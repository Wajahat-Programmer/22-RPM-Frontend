22-RPM-Backend

This is the backend for the 22-RPM project. It provides APIs for the frontend application to interact with the database and handle business logic.





 Prerequisites

Before running this project, make sure you have the following installed:

1. Node.js 
   Download from [https://nodejs.org](https://nodejs.org)

2. npm
   Verify with:
   ```bash
   npm -v
   node -v


   
git clone https://github.com/Wajahat-Programmer/22-RPM-Backend.git

cd 22-RPM-Backend

npm install


Create a .env file in the root directory.


Running the Server
nodemon start or node start
